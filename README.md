# Doosan M1013 ROS 2 Research Workspace

This repository contains my research workspace for the Doosan M1013 robotic manipulator using ROS 2 Humble.

The workspace is built on top of the official Doosan ROS 2 packages and includes simulation, visualization, and motion planning tools for collaborative robotic manipulation.

## Features

- ROS 2 Humble
- Doosan M1013 Robot
- Gazebo Simulation
- RViz2 Visualization
- MoveIt 2 Motion Planning
- ros2_control Integration
- Custom Robot Configuration
- Kuramoto-based Multi-Robot Synchronization
- B-spline Trajectory Generation
- Collision Detection and Avoidance
- Research-Oriented Development

## Base Repository

This workspace is developed using the official Doosan Robotics ROS 2 packages as the foundation, with additional modules for synchronization, trajectory generation, and collaborative robotic control.
